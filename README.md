# SpaWebDemo
基于ASP.NET CORE 2.0 + vue.js 实现的服务器和前端一体的框架工程

# 技术使用
1. 后端: Asp.net core / Entity Core / Mysql / Redis / REST API / Oracle ...
2. 前端: Vuejs / Webpack / Element-ui / Typescript / WebGL ...
3. 其他: Unity ...


# 环境
1. .net core sdk 2.0.2
2. nodejs v8.4.0


# 运行
1. cd SpaWeb
2. dotnet restore
3. npm install
4. dotnet run
